# 🚀 ServiceWala

A simple **service booking web app** where users can connect with local service providers.

---

## 🌐 Live Demo

👉https://chirag8533.github.io/Servicewala/

---

## ✨ Features

* 🔐 User Login & Registration
* 📊 Worker Dashboard
* 📅 Booking System
* 🔔 Toast Notifications
* 🔄 Simple Navigation (SPA-style)

---

## 🛠 Tech Stack

* HTML

* JavaScript
